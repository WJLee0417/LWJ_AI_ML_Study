# 🤖 LWJ AI & Machine Learning Study

> "데이터의 흐름을 이해하고, 지능적인 서비스를 구축하는 백엔드 개발자를 향하여"

이 저장소는 AI와 머신러닝(Machine Learning) 모델의 원리를 학습하고, 이를 실제 서비스에 적용하기 위한 실습 코드와 고민을 기록하는 공간입니다.

## 🎯 Study Purpose
단순히 주어진 AI 튜토리얼을 따라 치는 것을 넘어, 다음과 같은 목표를 달성하기 위해 학습합니다.

* **데이터 흐름의 완벽한 이해:** 원시 데이터가 전처리(Data Analysis) 과정을 거쳐 머신러닝/딥러닝(ML/DL) 모델을 통해 가치 있는 결과물로 변환되는 전체 파이프라인을 직접 경험하고 파악합니다.
* **AI 모델 Serving 및 백엔드 연동 고민:** Python으로 학습된 지능형 모델을 Java/Spring 기반의 백엔드 환경에 어떻게 효율적으로 연동하고, 사용자에게 안정적으로 서빙(Serving)할 수 있을지 아키텍처 관점에서 고민합니다.
* **시스템 최적화 및 알고리즘 역량 강화:** 내부 알고리즘(회귀, 분류, 신경망 등)이 수학적·논리적으로 어떻게 동작하는지 깊이 있게 파악하여, 더 나은 데이터 처리 로직과 최적화된 시스템 설계에 반영합니다.
