# 📊 01. Data Analysis (데이터 분석 및 전처리)

머신러닝/딥러닝 모델의 성능은 결국 '데이터의 품질'에서 결정됩니다. 
이 디렉토리에서는 원시 데이터(Raw Data)를 모델이 학습할 수 있는 형태로 정제하고 분석하는 과정을 실습합니다.

## 🎯 학습 목표
* **데이터 핸들링:** Pandas와 Numpy를 활용하여 대용량 데이터를 효율적으로 다루고 가공하는 방법을 익힙니다.
* **EDA (탐색적 데이터 분석):** Matplotlib, Seaborn 등을 이용해 데이터의 분포와 특징을 시각화하고 인사이트를 도출합니다.
* **DB 연동 관점:** 정제된 데이터를 향후 RDBMS(MySQL 등)와 어떻게 매핑하고 활용할지 데이터 파이프라인 관점에서 고민합니다.

## 📝 실습 내용
- Numpy 배열 연산 및 행렬 이해
- Pandas DataFrame을 활용한 결측치/이상치 처리
- 공공 데이터셋을 활용한 EDA 및 시각화 실습
