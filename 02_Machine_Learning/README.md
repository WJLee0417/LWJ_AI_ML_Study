# 🤖 02. Machine Learning (머신러닝 기초)

Scikit-learn을 활용하여 전통적인 머신러닝 알고리즘의 수학적 원리와 동작 방식을 이해하고 실습하는 공간입니다.

## 🎯 학습 목표
* **알고리즘 이해:** 회귀(Regression), 분류(Classification), 군집화(Clustering) 등 각 모델이 어떤 상황에서 최적의 성능을 내는지 파악합니다.
* **모델 평가 및 최적화:** 정확도(Accuracy), F1-Score 등의 평가지표를 이해하고, 하이퍼파라미터 튜닝을 통해 모델을 최적화하는 방법을 익힙니다.
* **논리적 사고:** 단순히 라이브러리를 호출하는 것을 넘어, 내부에서 어떤 수식과 로직으로 결과가 도출되는지 분석합니다.

## 📝 실습 내용
- 선형 회귀(Linear Regression)와 다항 회귀 실습
- 로지스틱 회귀(Logistic Regression) 분류 모델 구현
- 의사결정 나무(Decision Tree) 및 앙상블(Random Forest, XGBoost) 학습
- K-Means 클러스터링을 활용한 비지도 학습
